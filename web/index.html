Multilingual Virtual Assistant Mindfulness App

Designing a Simple, Accessible Mindfulness Assistant for Older Adults

Project Overview

The Multilingual Virtual Assistant Mindfulness App is a digital wellness tool designed to help older adults participate in guided mindfulness sessions. The application uses a conversational virtual assistant that walks users through short mindfulness exercises through a simple and accessible interface.

The system focuses on usability for older adults by emphasizing clear navigation, large readable text, and minimal interaction steps. The first version supports Korean while the architecture allows additional languages to be added later.

The project demonstrates how AI-guided digital assistants can support mental wellness programs while maintaining accessibility for aging populations.
